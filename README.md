## musicSurfer
***

## Description  
It queries data using a singular search term, then parses and combines that data into a list where you can quickly find the particular song;  from there, find further information about the track like lyrics, as well as a preview of the song or video.
***

## Libraries
* [Firebase](https://console.firebase.google.com/)  
* [Bootstrap](https://getbootstrap.com/)  
* [Jquery](https://jquery.com/)
* [particles.js](https://vincentgarreau.com/particles.js/)  
* [iTunes API](https://affiliate.itunes.apple.com/resources/documentation/itunes-store-web-service-search-api/) 
* [musixmatch API](https://developer.musixmatch.com/)
